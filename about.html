CATACOMB

~~~

CATACOMB is an isometric action survival game

where you play as a lone knight cursed to fend

off the unending hordes of the undead.

~~~

This project was completed in about 2 weeks

using vanilla JavaScript.

All of the art assets were modeled and

textured in Blender, a 3D modeling program.

~~~

For more info on technical implementation, please

check out the README on this project's Github repo.

~~~

~~~
